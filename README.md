# JFrivas - Página Web de Marketing Digital 🚀

Una página web moderna y responsive de marketing digital desarrollada con **HTML5**, **CSS3**, **JavaScript** y **Bootstrap 5**.

## 📋 Descripción

JFrivas es una página web profesional diseñada para servicios de marketing digital. Cuenta con un diseño moderno, responsivo y optimizado para todas las plataformas (desktop, tablet y móvil).

## ✨ Características

- **Diseño Responsivo**: Optimizado para dispositivos móviles y escritorio
- **Navbar Interactivo**: Menú de navegación con colapso en dispositivos pequeños
- **Carrusel de Imágenes**: Slider automático con controles de navegación
- **Formulario de Suscripción**: Integración para captar correos de potenciales clientes
- **Sección de Equipo**: Muestra el equipo profesional
- **Sección de Contacto**: Información de contacto directo
- **Tipografía Dinámica**: Efecto typewriter con texto animado
- **Iconografía Moderna**: Uso de Font Awesome 5 para iconos profesionales

## 🛠️ Tecnologías Utilizadas

- **HTML5** - Estructura semántica
- **CSS3** - Estilos y diseño responsivo
- **JavaScript** - Interactividad y animaciones
- **Bootstrap 5** - Framework de CSS responsivo
- **Font Awesome 5** - Librería de iconos
- **Google Fonts** - Tipografías personalizadas (Montserrat, Roboto)

## 🎨 Personalización

Puedes personalizar los siguientes elementos:

1. **Colores**: Modifica las variables de Bootstrap en `style1.css`
2. **Imágenes**: Reemplaza las imágenes en la carpeta `img/`
3. **Texto**: Actualiza el contenido en `index.html`
4. **Tipografía**: Cambia las fuentes en el CSS
